# Blockchain Transaction Tracker

Real-time Ethereum blockchain tracker demonstrating Web3.js integration and blockchain data visualization.

## Features

- **Real-time Block Monitoring**: Live updates of latest blocks
- **Web3.js Integration**: Direct blockchain interaction
- **Balance Checker**: Query any Ethereum address balance
- **Transaction History**: View recent transactions
- **Network Stats**: Gas prices, difficulty, and more
- **Data Visualization**: Chart.js for transaction volume

## Technologies

- Web3.js
- Chart.js
- MetaMask integration
- Infura/Alchemy RPC
- Vanilla JavaScript

## Setup

1. Install MetaMask browser extension OR get an Infura API key
2. If using Infura, replace the API key in app.js
3. Open index.html in a browser

## Skills Demonstrated

- Blockchain interaction with Web3.js
- Asynchronous API calls
- Real-time data polling
- Address balance queries
- Transaction parsing
- Data visualization
